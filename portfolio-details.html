Sky Things (Plataforma IoT)

Este es hasta ahora uno de mis proyectos más grandes y ambiciosos. La idea es crear una herramienta web que permita simplificar la integración del hardware y el software para ayudar a los desarrolladores a implementar soluciones IoT de forma rápida y sencilla.

  • Website: ieiot.com.mx/appweb
En conjunto con un equipo de 4 personas cubriendo el área de mecatrónica, electrónica y software diseñamos y creamos placas electrónicas que se conectaran a esta herramienta la cual llamamos Sky Things.

Para crear esta plataforma utilizamos en el backend, php bajo el framework de codeigniter y MySQL en las primeras versiones, posteriormente migramos a MongoDB. Para el frontend JavaScrip usando el framework de VueJs.

La comunicación de los dispositivos fue por MQTT utilizando los servicios de un broker, en específico EMQX. La versión beta la lanzamos sobre una VPS de AWS.
Lo que te permite hacer esta herramienta es:

  • Crear roles de usuario (Administrador e invitado).

  • Registrar el lugar donde estará instalado el dispositivo IoT.

  • Seleccionar el tipo de dispositivo, hasta ahora está diseñado para tres:

    • Real time: se visualiza como una casilla que muestra datos enviados del dispositivo en tiempo real.
    • Swicht: es un botón que te permite mandar acciones a un dispositivo.
    • Slider: una barra que te perite variar intensidades.

  • Creación de invitados para que puedan manipular los dispositivos instalados en tu perfil

  • Conceder y revocar los permisos a los invitados.
Liberamos el código de esta versión para que puedan usarlo sin fines de lucro.
